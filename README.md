# ship_model_collections
This is a collections of ship manoeuvring model that has been varified in the harbour shipping engineering. The pilots and captains agree that these models are corresponding with the real ship motion. These model includes the large bulk, container, and the tanker.
The container ship manoeuvring model with twin propeller twin rudder(TPTR) is presented here. This model is MMG struct, the main parameters are set by system identification.
As the simulink has been used in phD study, and the other tools and wind-current model have been coding with matlab, the collections are programming by matlab.
